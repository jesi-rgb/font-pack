Font Case

How does Font Case work?

Font Case works automatically upon installing. You just surf the web, as usual, and Font Case keeps track of every font you run into.

All the different fonts and the sites you found them will be gathered in here.

Same fonts found on same sites will be grouped together. Same fonts found in different sites will not be grouped together.

The list is sorted in inverse chronological order; last found, first.

If the font is installed in your computer, it will be displayed properly; otherwise we fall back to our old trusty Helvetica.
